# FKT v0.3.0 — V2 native P2WSH multisig cosign

**Date:** 2026-07-10  
**Product string:** `0.3.0`  
**Scope:** Classic bare `OP_CHECKMULTISIG` **native P2WSH** cosign + finalize.

## What shipped

| Capability | Behavior |
|------------|----------|
| Cosign | Your seed’s key → BIP143 + DER → `PSBT_IN_PARTIAL_SIG` |
| Multi-deriv match | Walk **all** input BIP32 derivations; use the path whose pubkey matches this seed (required for m-of-n) |
| Finalize | When ≥ *m* partials present → final witness `[OP_0][sigs…][witnessScript]` |
| Under threshold | Leaves partials; status `[PARTIAL]` (not fully signed) |
| Preview | Shows `m-of-n cosign` on P2WSH inputs when script is parseable |
| Parse | Accepts `PARTIAL_SIG` (0x02) on re-load (second cosigner) |

## Automated proof

```bash
cd cli
make test-multisig
# 1of1 finalize · 2of2 A→partial B→finalize · 2of3 A+B finalize · wrong seed reject
```

## Explicit non-goals (still)

- MuSig2 / Taproot multi / FROST  
- Miniscript / `multi_a`  
- Nested P2SH-P2WSH multi (possible later)  
- Holding more than one cosigner seed on one Ice Cold run
