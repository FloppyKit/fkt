## Added / changed

- **Parse:** BIP174 proprietary type `0xFC` accepted on **global**, **input**, and **output** maps (length-capped; not interpreted).
- **Sign:** no longer strips output proprietary keys; input/global `0xFC` remain in the buffer through sign.
- Fixture: `cli/tests/sparrow-real/unsigned/p2wpkh_proprietary_passthrough.psbt`  
  (dummy bark-shaped keys on global + input + output).
- Harness: `check_proprietary` verifies every unsigned `0xFC` key/value survives in fkt-signed.
