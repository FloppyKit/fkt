KNOWN GOTCHAS
-------------
- DOS 8.3: use FKTSIGN.EXE (FKTSIGNER.EXE = illegal command)
- DJGPP stub requires CWSDPMI.EXE in same directory
- Mount full path in DOSBox (no ~); dir before run to confirm files on C:
- Mac Homebrew: brew install dosbox-x (command is dosbox-x)
- M4 build: Rosetta + arch -x86_64 for i586-pc-msdosdjgpp-gcc
- SSH to X280: DISPLAY=:0 needs xhost +local: or run DOSBox locally on laptop
